# GymApp

GymApp es una aplicación web para **crear y organizar rutinas de entrenamiento de forma visual**, pensada principalmente para entrenadores personales y usuarios que trabajan con programas estructurados.

El foco del proyecto está en la **planificación clara y rápida** de rutinas, permitiendo construir programas completos a partir de una biblioteca de ejercicios reutilizable.

---

## Qué permite hacer

- Crear programas de entrenamiento con varios días
- Asignar ejercicios a cada día mediante un planner visual
- Definir series, repeticiones, descansos, tempo e intensidad
- Reutilizar ejercicios en distintos programas
- Organizar ejercicios por categorías (grupo muscular, equipamiento, tipo)

---

## Demo

La versión demo incluye:
- Biblioteca de ejercicios precargada
- Varios programas de ejemplo con distintos enfoques
- Planner funcional en modo lectura

Algunas acciones están deshabilitadas intencionadamente en la demo.

---

- Frontend: React
- Arquitectura orientada a dominio
